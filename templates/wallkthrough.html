Walkthrough

It is a New Year puzzle to entertain friends

There are three levels of difficulty. Every next one is more difficult but also a way more fun.

Lvl 0

The main thing that attracts attention is the cube. Of course, it is the Rubik's cube.

Also, there are control buttons. Maybe you are already familiar with the rotation notation, or maybe you could click on them and notice which part of the cube is rotating and in which direction

If solving a cube ('a cube' because you can reroll it if you clear browser cookies or use an incognito mode) is hard by itself, then you can use a hint.

A good-old-trick is to see what's there on the page but hardly visible. Just Ctrl+a. You will notice there is a 'hint' down there. If you press it you will see the cube rotations sequence.

Well... If you can't do Rubik's cube 'the usual way', if you know how it was scrambled you can unscramble it by doing things in reverse order.

I assume at this point you have unscrambled it. Well done!

Lvl 2

But what's that strange mentioning 'the url is a base64 encoded something'? If you haven't noticed that already ('==' at the end of a string is a solid sign of this encoding), the 'MjAyMg==' is an encoded string.

So let's decode it!

There are a lot of ways to do it. Arguably, the most quick one is to use some online decoder. Something like asciitohex.com.
